# Design — Décisions d'architecture (ADR)

Un **Architecture Decision Record (ADR)** documente une décision technique structurante : le contexte qui l'a motivée, les options envisagées, la décision retenue et ses conséquences. Chaque ADR est immuable une fois accepté — il capture l'état de la réflexion à un instant donné.
